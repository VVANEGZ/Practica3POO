# Práctica POO: Simulación de Cajero Automático
Este repositorio contiene la solución a una práctica de Programación Orientada a Objetos de la materia de Diseño y POO. 

## Aviso Importante sobre las Ramas (Branches)
Este repositorio utiliza dos ramas para el seguimiento del desarrollo:

master: Esta rama contiene la primera versión de la práctica. Dicha versión puede no cumplir con todos los criterios, así que quise volverlo hacer en la rama Main.

main: Esta es la versión final, corregida y mejorada del proyecto. Se solucionaron los errores presentes en master.

Para revisar el código funcional y la solución final, por favor asegúrate de estar en la rama main.

### ¿Cómo ejecutar el programa?
Asegúrate de estar en la rama main para usar la versión funcional, pero si solo quiere la versión que se realizó en clase, use la rama Master.

### Clonar el repositorio:

```Bash

git clone https://github.com/VVANEGZ/Practica3POO.git
git checkout main
```
